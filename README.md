# NeighbourNet
# 🌐 NeighbourNet AI Ultimate

NeighbourNet AI Ultimate is a modern full-stack community networking platform that helps people connect, learn, collaborate, and support each other locally.

It combines:
- AI mentor suggestions
- Real-time community chat
- Emergency help requests
- Nearby community discovery
- Skill networking
- Leaderboards
- Community analytics
- Resume builder concepts
- Modern animated UI
